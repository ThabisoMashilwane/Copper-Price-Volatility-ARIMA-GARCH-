# Copper-Price-Volatility-ARIMA-GARCH-
ARIMA–GARCH Analysis of Global Copper Price Volatility (1990–2025) Third-year econometric time series project modelling copper prices using ARIMA and GARCH-family models in R. The study examines volatility clustering, non-normality, ARCH effects, model selection (AIC/BIC), and conditional variance forecasting.
